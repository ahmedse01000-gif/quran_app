# تطبيق القرآن الكريم (Quran App)

تطبيق جوال متكامل لقراءة وسماع القرآن الكريم مع مميزات متقدمة وواجهة مستخدم جميلة وسهلة الاستخدام.

## المميزات ✨

- 📖 **قراءة القرآن الكريم**: اقرأ جميع السور القرآنية بنصوص واضحة وسهلة القراءة
- 🎵 **الاستماع للتلاوات**: استمع إلى تلاوات متعددة من قراء مختلفين
- 🔖 **الإشارات المرجعية**: احفظ آياتك المفضلة للعودة إليها لاحقاً
- 🌙 **الوضع الليلي**: استمتع بالقراءة في الليل مع وضع ليلي مريح للعينين
- 🔍 **البحث**: ابحث بسهولة عن السور التي تريدها
- 💾 **التخزين المحلي**: احفظ إعداداتك وإشاراتك المرجعية محلياً
- 🎨 **تصميم جميل**: واجهة مستخدم حديثة وجميلة

## المتطلبات 📋

- **Flutter**: إصدار 3.11.0 أو أحدث
- **Dart**: إصدار 3.11.0 أو أحدث
- **Android SDK**: API Level 21 أو أحدث
- **Java**: JDK 17 أو أحدث

## التثبيت 🚀

### 1. استنساخ المستودع
```bash
git clone https://github.com/yourusername/quran_app.git
cd quran_app
```

### 2. تثبيت التبعيات
```bash
flutter pub get
```

### 3. تشغيل التطبيق
```bash
flutter run
```

### 4. بناء APK
```bash
flutter build apk --release
```

### 5. بناء AAB (للنشر على Google Play)
```bash
flutter build appbundle --release
```

## الاستخدام 📱

### الشاشة الرئيسية
- عرض جميع السور القرآنية
- البحث عن السور
- الانتقال إلى شاشة القراءة

### شاشة القراءة
- عرض آيات السورة
- إضافة/حذف الإشارات المرجعية
- الانتقال إلى شاشة مشغل الصوت

### شاشة مشغل الصوت
- اختيار القارئ المفضل
- تشغيل/إيقاف التلاوة
- التحكم في مستوى الصوت

### شاشة الإشارات المرجعية
- عرض جميع الإشارات المرجعية المحفوظة
- حذف الإشارات المرجعية
- الانتقال إلى الآية المرجعية

### شاشة الإعدادات
- تبديل الوضع الليلي/النهاري
- ضبط حجم الخط
- ضبط ارتفاع السطر
- عرض معلومات التطبيق

## API المستخدم 🌐

يستخدم التطبيق [API القرآن الكريم](https://alquran.cloud/api) لجلب البيانات:

- **Base URL**: `https://api.alquran.cloud/v1`
- **السور**: `/surah`
- **السورة المحددة**: `/surah/{number}`
- **التلاوات**: `/surah/{number}/{reciter}`
- **الإصدارات**: `/edition`
